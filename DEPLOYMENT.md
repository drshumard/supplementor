# Dr. Shumard Protocol Manager — Deployment Guide

## Server: Ubuntu + PM2 + Nginx + MongoDB
**Domain:** `fm.drshumard.com`  
**Ports:** Backend `8047`, Frontend `3847`

---

## Prerequisites

Your server should already have:
- Node.js 18+ (`node -v`)
- Python 3.11+ (`python3 --version`)
- MongoDB running (`systemctl status mongod`)
- Nginx running (`systemctl status nginx`)
- PM2 installed (`pm2 --version`, if not: `npm install -g pm2`)
- Certbot installed (`certbot --version`)
- Yarn installed (`yarn --version`, if not: `npm install -g yarn`)

## First-Time Setup

```bash
# 1. Clone the repo
cd /var/www
git clone <your-repo-url> drshumard-protocol
cd drshumard-protocol

# 2. Make deploy script executable
chmod +x deploy.sh

# 3. Run deploy
sudo ./deploy.sh

# 4. Add SSL
sudo certbot --nginx -d fm.drshumard.com

# 5. PM2 startup (so it survives reboots)
pm2 startup
pm2 save
```

## Updating (Redeployment)

Just run:
```bash
cd /var/www/drshumard-protocol
sudo ./deploy.sh
```

This will:
1. `git pull` latest code
2. Install any new Python/Node dependencies
3. Rebuild the React frontend
4. Restart PM2 processes
5. Reload Nginx config

## Architecture

```
Internet
  │
  ▼
Nginx (port 80/443) — fm.drshumard.com
  ├── /api/*  → FastAPI (127.0.0.1:8047) via PM2
  └── /*      → React build (127.0.0.1:3847) via PM2
                    │
                    ▼
              MongoDB (localhost:27017)
              Database: drshumard_protocol
```

## Configuration Files

| File | Purpose |
|------|--------|
| `/var/www/drshumard-protocol/backend/.env` | Backend secrets (JWT, MongoDB, Google Drive) |
| `/var/www/drshumard-protocol/ecosystem.config.js` | PM2 process config |
| `/etc/nginx/sites-available/fm.drshumard.com` | Nginx reverse proxy |

## Common Commands

```bash
# Check status
pm2 status

# View logs
pm2 logs drshumard-protocol-backend
pm2 logs drshumard-protocol-frontend
pm2 logs  # all logs

# Restart
pm2 restart drshumard-protocol-backend
pm2 restart drshumard-protocol-frontend
pm2 restart all

# Stop
pm2 stop all

# MongoDB shell
mongosh drshumard_protocol
```

## Backups

```bash
# Backup MongoDB
mongodump --db drshumard_protocol --out /backup/$(date +%Y%m%d)

# Restore
mongorestore --db drshumard_protocol /backup/YYYYMMDD/drshumard_protocol/
```

## Troubleshooting

```bash
# Backend won't start
pm2 logs drshumard-protocol-backend --lines 50

# Check if ports are in use
lsof -i :8047
lsof -i :3847

# Nginx config test
nginx -t

# Check MongoDB
systemctl status mongod

# Full restart
pm2 delete all
cd /var/www/drshumard-protocol
./deploy.sh
```

## DNS Setup

Point `fm.drshumard.com` to your server's IP:
```
Type: A
Name: fm
Value: <your-server-ip>
TTL: 300
```

After DNS propagates, run `sudo certbot --nginx -d fm.drshumard.com` for SSL.
